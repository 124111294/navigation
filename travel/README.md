# 烟雨江南

## 项目概述
烟雨江南是一个**纯静态网页**项目，使用 **HTML**、**CSS**、**JQuery** 和 **JQuery 插件** 实现所有功能。该项目为游客提供一个集成旅游服务的互动平台。虽然没有后端支持，但网站依然能够提供丰富的用户体验。主要功能包括：

- **登录注册**：模拟用户登录和注册，保存游客信息（基于前端存储）。
- **美食推荐**：展示江南本地特色美食，游客可以查看和选择推荐的餐厅。
- **住宿推荐**：提供江南地区的住宿信息，帮助游客选择合适的酒店或民宿。
- **景点推荐**：为游客提供热门景点的详细介绍和推荐，帮助规划旅游路线。
- **目的地推荐**：根据游客偏好推荐不同的旅游目的地，帮助游客发现新的旅游地点。
- **游客打卡分享**：游客可以通过前端界面查看和分享旅游打卡照片、心得等。
- **旅游跟团买票**：提供虚拟的跟团旅游服务和门票展示，用户可以查看不同旅游套餐信息。

## 技术栈
- **前端技术**：
  - **HTML5**：构建网站页面的基础结构。
  - **CSS3**：负责页面的样式设计，确保响应式布局，支持各种设备和屏幕尺寸。
  - **JQuery**：简化 JavaScript 操作，提高页面交互性和动态效果。
  - **JQuery 插件**：用于增强网站功能等。

## 网站功能
### 登录注册
尽管没有后端支持，用户可以在前端模拟登录和注册过程，系统通过浏览器的本地存储保存用户信息。

### 美食推荐
展示江南地区的特色美食推荐，游客可以通过列表查看餐馆和餐饮类型。

### 住宿推荐
提供多种住宿选择，用户可以浏览不同类型的住宿选项，如酒店、民宿等，且可通过前端动态显示不同价格和服务信息。

### 景点推荐
推荐江南地区的知名景点，提供简单的描述和图片。游客可以根据自己的兴趣选择景点，并查看基本信息。

### 目的地推荐
根据游客的偏好（如自然景观、历史文化等），通过前端页面动态推荐最适合的旅游目的地。

### 游客打卡分享
游客可以通过平台的图片展示功能查看其他游客的打卡分享，分享自己旅游时的照片和心得，增加互动性。

### 旅游跟团买票
提供虚拟的旅游跟团服务和门票信息展示，用户可以选择自己感兴趣的旅游套餐，查看具体内容。

## 关于网站
烟雨江南旨在为游客提供一个完全静态但功能丰富的旅游平台，尽管没有后端服务器的支持，网站依然可以通过前端技术实现基本的互动功能。游客可以通过本平台获取到关于江南的各种旅游信息，并与其他游客分享旅行经历。

网站涵盖了从美食到住宿，从景点到打卡分享的多种功能，帮助游客在没有复杂登录系统的情况下，轻松享受虚拟的江南旅游体验。通过使用 **HTML**、**CSS** 和 **JQuery** 技术，网站页面简洁且富有互动性，极大地提高了用户的浏览体验。
